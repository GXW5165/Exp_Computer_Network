# Computer Network Lab

## 📌 项目简介

本仓库用于存放《计算机网络》课程实验内容与实现代码。本课程实验采用**自顶向下的分层设计方法**，从应用层逐步深入到底层协议，通过一系列实践任务，帮助学生系统掌握计算机网络体系结构及其核心机制。

实验围绕真实网络通信流程展开，涵盖 Socket 编程、TCP/IP 协议栈实现、路由与交换等关键内容，强调理论与实践的结合。

---

## 🧭 实验体系结构

本课程实验按照计算机网络五层模型进行组织：

- 网络通信的完整过程
- 各层协议的功能与实现
- 不同层之间的协作关系

---

## 🧪 实验内容概览

本课程共包含九个实验，具体如下：

| 实验编号 | 实验名称             | 所属层次     | 主要内容 |
|----------|----------------------|--------------|----------|
| 实验一   | Socket应用编程       | 应用层       | 基于 Socket API 实现客户端与服务器通信 |
| 实验二   | Echo传输             | 传输层       | 实现简化 TCP 协议栈，完成基本通信 |
| 实验三   | 文件传输             | 传输层       | 基于 TCP 实现文件传输功能 |
| 实验四   | 可靠传输             | 传输层       | 实现重传、滑动窗口等机制 |
| 实验五   | 路由器转发           | 网络层       | 实现 IP 数据包转发 |
| 实验六   | 高效IP路由查找       | 网络层       | 优化路由查找算法 |
| 实验七   | mOSPF                | 网络层       | 实现动态路由协议 |
| 实验八   | 广播网络             | 数据链路层   | 实现广播机制 |
| 实验九   | 交换机转发           | 数据链路层   | 实现二层交换功能 |

---

## 🎯 实验目标

通过本课程实验，学生将掌握：

- Socket 编程模型与网络通信流程
- TCP/IP 协议栈的核心机制
- 数据封装与分层设计思想
- 网络协议的实现方法与调试技巧
- Mininet 等网络仿真工具的使用

---

## 🛠 实验环境

推荐实验环境如下：

- 操作系统：Linux（Ubuntu / Mininet VM）
- 编程语言：C
- 工具：
  - GCC / Make
  - Mininet
  - tcpdump（抓包分析）
